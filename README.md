# FastAPI

## Requirements

Python 3.6+

## Installation

``` shell script
pip install fastapi
pip install uvicorn[standard]
```

## Run it

`uvicorn main:app --reload`

http://127.0.0.1:8000

http://127.0.0.1:8000/items/1

http://127.0.0.1:8000/items/1?q=somequery

## Generated docs

http://127.0.0.1:8000/docs
